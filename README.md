# CFG Ambiguity Detector

An interactive web-based simulation that detects ambiguity in Context-Free Grammars (CFG) by generating and visualizing parse trees. It helps users understand how multiple derivations can exist for the same input string.

---

## 🚀 Features

* Parse tree generation using D3.js
* Ambiguity detection (unique tree comparison)
* Step-by-step animation (Play mode)
* Optimized parsing (pruning + memoization)
* Clean modern UI (Tailwind CSS)

---

## 🧠 How It Works

* Input a grammar and a string
* The system generates parse trees
* Duplicate trees are removed
* Result:

  * 0 trees → Invalid
  * 1 tree → Unambiguous
  * 2+ trees → Ambiguous
